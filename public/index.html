RapidLink

A simple, fast, and custom URL shortener.

1. Setup Your Domain

Add your custom domain to start creating short links. You only need to do this once.

DNS Configuration

To point your domain to this service, create an A record in your DNS provider's settings and point it to the following Vercel IP address:

76.76.21.21

DNS changes can take some time to propagate. This is a standard Vercel IP for Hobby/Pro plans.
